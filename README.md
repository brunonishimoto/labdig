# Laboratório Digital II
This repository contains the projects developed in *PCS3645 - Laboratório Digital II.*

## Usage
First create a project in Quartus 16.2 importing the files '.vhd' and '.vwf'. **Be sure to name your project equals the top level entity**.
Next, go to Assignments -> Import Assignment and choose the '.qsf' file (this is to import pin planner, so you don't have to redo it).
